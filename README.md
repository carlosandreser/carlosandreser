# 🏦 Sistema Bancário em C

Este projeto consiste em um sistema de gerenciamento bancário desenvolvido inteiramente em **C**. O objetivo principal é aplicar conceitos fundamentais de Engenharia de Software, como modularização, manipulação de arquivos e tipos abstratos de dados.

## 🚀 Funcionalidades
- **Cadastro de Clientes:** Registro de novas contas com validação.
- **Operações Financeiras:** Depósitos, saques e transferências entre contas.
- **Consulta de Saldo:** Visualização rápida do extrato e saldo atual.
- **Persistência de Dados:** Armazenamento das informações em arquivos para que os dados não sejam perdidos ao fechar o programa.
- **Segurança Simples:** Implementação de lógica para controle de acesso.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** C (Padrão C11/C17)
- **Compilador:** GCC
- **Ambiente:** VS Code / Terminal

## 📂 Estrutura do Projeto
O projeto foi estruturado seguindo boas práticas de organização:
- `main.c`: Ponto de entrada do programa e fluxo principal.
- `banco.h`: Definição de structs e protótipos de funções.
- `banco.c`: Implementação da lógica de negócio e operações.
- `dados/`: Pasta destinada aos arquivos `.txt` ou `.bin` de armazenamento.
